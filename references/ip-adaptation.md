# FinalRound IP / Genre Adaptation

把电影、剧集、游戏、历史仪式、互联网梗的镜头语法，转译成 FinalRound 可用的 15 秒竖屏广告结构。不要复制 IP，不要复刻角色，不要把原片名、角色名、演员名、logo、台词或可读引用写进生成 prompt。

## 使用条件

用户出现以下表达时读取本文件：

- "像 Matrix / 1984 / Squid Game / Oppenheimer / Godfather / Napoleon coronation / MrBeast / Blade Runner / LOTR"
- "用某个电影/剧集/游戏的感觉"
- "把这个新闻做成某种经典场面"
- "宗教仪式、加冕、审判、控制室、挑战赛、黑帮权力、威权系统、灾难片、赛博朋克"

## 核心原则

FinalRound 不是泛 meme 账号。IP 转译只有在能接到 interview / offer / job market / layoff 后下一份工作 / AI hiring filter / candidate survival 时才成立。

不要问"这个像哪个 IP"。先问：

1. 谁在掌握规则？
2. 谁在被筛选、审判、淘汰或追问？
3. 候选人为什么需要 FinalRound 这个 edge？
4. FinalRound reveal 是改变权力关系，还是解释赢家为什么稳？

如果答不上第3和第4题，这个素材先标记为 `未接产品 / 待转化`，不要硬写成广告。

## 工作流

1. **抽结构，不抽皮肤**：把 IP 拆成权力关系、空间规则、镜头语法、声音节奏、关键动作，不保留原片名和角色。
2. **接产品场景**：把结构接到真实面试、offer竞争、裁员后下一场面试、AI筛选或求职者临场表达。
3. **选择安全模板**：用下方的 FinalRound-safe template label，而不是原 IP 名字。
4. **设计证据镜头**：至少两个 close-up 证明状态差，例如眼神、手、屏幕反应、offer信封、系统扫描、面试官沉默。
5. **压到15秒**：0-3秒给规则或冲突，3-9秒加压，9-12秒预备 reveal，12-15秒 FinalRound 一句短话收住。
6. **写安全边界**：禁止原片名、角色名、演员名、logo、新闻台标、可读引用、真实背书感。

## 转译样式

不要写：

```
Make it like Squid Game. The recruiter is the Front Man. Use red light green light.
```

改写成：

```
15秒竖屏9:16，harmless interview elimination game skit。一个冷白灯光的 hiring arena，候选人排队坐在笔记本前，系统广播宣布每个人只有一次回答机会。不要出现任何真实剧集名、角色名、面具、logo或可读文字。
```

不要写：

```
Matrix-style office scene with Neo escaping the simulation.
```

改写成：

```
15秒竖屏9:16，AI job market control room parody。冷蓝控制室里，系统角色通过监控墙筛掉求职者；主角站在人工天空边界前，听到下一场真实面试开始，用一句低声 FinalRound reveal 反抗系统。
```

## IP 转 FinalRound 的三种落点

### 1. Hidden Rule Reveal

适合：1984、Matrix、elite recruiting、AI fellowship、founder parody。

结构：权力角色讲规则 → 候选人意识到普通准备不够 → 权力角色或赢家揭示 hidden rule → `Smart people use FinalRound AI.`

关键：FinalRound 是规则，不是广告口号。

### 2. Winner Secret Reveal

适合：Godfather、coronation、offer envy、campus/club winner girls、public news walkout。

结构：别人祝贺/围观赢家 → 旁人不服或记者追问 → 镜头靠近赢家表情/道具 → 赢家低声说出 FinalRound。

关键：前10秒先证明赢家的 status，最后5秒才解释为什么。

### 3. System Escape

适合：Matrix、Squid Game、layoff game、AI job market、dome escape。

结构：系统宣布淘汰/筛选/岗位消失 → 主角即将被规则吞掉 → 主角用 FinalRound 稳住下一场面试 → offer 或面试表现反转。

关键：FinalRound 是逃出系统的工具，不是对系统的空泛评论。

## 提示词护栏

每个 IP-adapted prompt 末尾加：

```
Do not show or mention any original film, TV, game, character, actor, studio, logo, title card, news logo, readable quote, or recognizable copyrighted dialogue. This is a parody-style original job interview skit using only abstract genre grammar. No subtitles, no captions, no readable UI, no product UI, no end card.
```

如果使用真实人物或 R2 reference，再加：

```
Use Reference [n] only as the character identity reference. Do not show the real person's name in dialogue, visible text, UI, badge, logo, or title card. No endorsement, no real news footage.
```

## 自检

- 是否已从 IP 名称转成 safe template label？
- 是否没有在生成 prompt 里保留原片名、角色名、演员名、logo、台词？
- 是否接到了 interview / offer / job market / candidate survival？
- 是否有两个证据特写？
- FinalRound reveal 是否在剧情里有功能，而不是硬广？
- 如果还没有产品落点，是否明确标成 `未接产品 / 待转化`？
